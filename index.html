Applications Disponibles

Gestion d'Emploi du Temps

Application complète de gestion scolaire

Une application web complète permettant de créer, gérer et visualiser les emplois du temps pour les établissements scolaires.

Fonctionnalités principales

  • Répartition automatique des horaires
  • Saisie manuelle des cours
  • Visualisation par enseignant ou classe
  • Exportation et importation des données
  • Interface d'administration sécurisée
Visiter le projet

Formulaire de Commande

Gestion des commandes de matériel scolaire

Un formulaire intuitif permettant aux établissements scolaires de commander facilement leur matériel pédagogique.

Fonctionnalités principales

  • Sélection par cycle (Collège/Lycée)
  • Calcul automatique des quantités
  • Récapitulatif détaillé de commande
  • Envoi direct via WhatsApp
  • Interface responsive et intuitive
