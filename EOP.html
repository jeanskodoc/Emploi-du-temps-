Programmation Automatique des DEVOIR Surveillés

Générez automatiquement la programmation des devoirs pour les classes avec répartition des surveillants.

Nouvelle fonctionnalité : Toutes les classes composeront dans une matière le même jour, mais pas nécessairement la même matière pour chaque classe.

Durée des DEVOIR : Collège = 2h, Lycée = 3h

Éviter les répétitions : Une même matière ne sera pas répétée le même jour pour différentes classes.

Éviter les répétitions dans la même classe : Une même matière ne sera pas répétée dans la même classe pendant toute la période de programmation.
