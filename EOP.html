Programmation Automatique des DEVOIR Surveillés

Générez automatiquement la programmation des devoirs pour les classes avec répartition des surveillants.

Principe : Toutes les classes composent chacune une matière le même jour, même si ce n'est pas la même matière dans toutes les classes.

Durée des DEVOIR : Collège = 2h, Lycée = 3h

Pas de répétition intra-classe : Une même matière ne sera pas répétée dans la même classe pendant toute la période de programmation (sauf si toutes les matières ont déjà été utilisées).

Matières différentes possibles : Deux classes différentes peuvent composer la même matière le même jour.

Configuration de la Programmation

Maintenez Ctrl (ou Cmd) pour sélectionner plusieurs classes

Conseils d'utilisation

  • Configurez d'abord les matières par classe avant la répartition automatique.
  • Le mercredi après-midi est réservé (aucun cours).
  • Les mardi et vendredi après-midi sont réservés aux DEVOIR uniquement.
  • Utilisez l'export régulier pour sauvegarder vos données.
  • La détection de conflits d'enseignants est automatique lors de la saisie.
